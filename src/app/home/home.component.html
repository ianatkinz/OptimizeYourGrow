Optimize your grow

Given the outdoor temperature, humidity and parameters of your growing facility, this program determines the optimial quantity of ventilation, heating, cooling, and evaporative cooling. To get started, click on one of the tools above.

About

Optimizeyourgrow was built by Ian Atkins. The optimization program is written in Julia. The web interface was built using using Angular. The interface and the linking between the Julia code and the interface is based on open source code for DifferentialEquations.jl.
